# retarget_with_empties
Retarget Rig using empties method
retarget animation to your rig, using other animated rig that may have different proportions.
based on this:
https://www.youtube.com/watch?v=2GSzn7qkE7Y
Addon also contains automatic bones chain detection - meaning it will try to automatically detect:
 - spine chain
 - arms chain-
 - legs chains 
 etc.
